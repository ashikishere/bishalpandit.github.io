Projects

OxyFinder

OxyFinder helps to find genuine oxygen suppliers. We basically store the data of oxygen suppliers in a database and display them. Any user visiting the site can also add oxygen suppliers through a form.

VaxSlot

VaxSlot makes it possible to find the nearest vaccine centres for people. Users need to provide the PIN Code of their locality. The web application uses the Cowin API to fetch vaccine centres data and display it to users.

Weather App

A simple weather app which tells us a bunch of things other than just temperature. The user needs to enter the location of any place. Openweather API works in the background to provide correct data.
